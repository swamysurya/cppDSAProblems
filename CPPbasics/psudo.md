## Reading Input from User

```
INPUT name
```

## Print Variable

```
OUTPUT name
(or)
PRINT name
```

## Variable Initialization

```
SET name TO "surya"
```

## IF Statement

```
IF score < 40 THEN
 PRINT "pass"
END IF
```

## IF-ELSE Statement

```
IF score < 40 THEN
 PRINT "pass"
ELSE
 PRINT "fail"
END IF
```

## IF-ELSE IF-ELSE Statement (Else-If Ladder)

```
IF score < 40 THEN
 PRINT "pass"
ELSE IF score < 60 THEN
 PRINT "good"
ELSE
 PRINT "super"
END IF
```

## FOR Loop

```
FOR index FROM 0 TO 10
 PRINT index
END FOR
```

## WHILE Loop

```
SET i TO 0
WHILE (i <= 5)
 PRINT i
 SET i TO i + 1
END WHILE
```

## Functions / Procedures

```
PROCEDURE greet(name)
 PRINT "hello" + name
END PROCEDURE
```

## List Initialization and Accessing

```
SET nums TO [10, 20, 30, 40]
PRINT nums[1] // here we are using pos-based indexing
```
